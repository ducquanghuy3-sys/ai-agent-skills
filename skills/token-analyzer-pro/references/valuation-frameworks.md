# Crypto Valuation Frameworks

## 1. Revenue Multiple (P/S Ratio)
Best for: DeFi protocols with fee revenue (Uniswap, Aave, Lido)
Formula: FDV / Annualized Revenue
Benchmarks: L1s: 50-200x | DeFi Blue-chip: 10-50x | Emerging DeFi: 5-20x

## 2. TVL-Based (Market Cap / TVL)
Best for: DeFi lending/DEX protocols
Formula: Market Cap / Total Value Locked
Benchmarks: >1.0 = potentially overvalued | 0.3-1.0 = fair | <0.3 = potentially undervalued
Caveat: TVL can be gamed with incentives; normalize for organic TVL

## 3. Active User Valuation
Best for: L1/L2 chains, social protocols, gaming
Formula: FDV / Daily Active Addresses
Benchmarks: Compare within same category only (L1 vs L1, not L1 vs DeFi)

## 4. Fee Revenue Yield
Best for: Any protocol distributing revenue to token holders
Formula: Annual Fee Revenue to Holders / Market Cap × 100
Benchmarks: >5% = strong yield | 1-5% = moderate | <1% = minimal direct value
Compare to: ETH staking yield (~3-4%) as risk-free crypto rate

## 5. NVT Ratio (Network Value to Transactions)
Best for: L1 payment/transfer chains
Formula: Market Cap / Daily On-chain Transaction Volume
Interpretation: High NVT = overvalued relative to usage | Low NVT = undervalued

## 6. Developer Activity Score
Proxy for long-term value creation
Metrics: GitHub commits, active contributors, repos, PRs merged (90-day window)
Source: Electric Capital Developer Report, GitHub API
Top tier: >100 monthly active devs | Strong: 20-100 | Weak: <20

## 7. Meme / Narrative Token Framework
For tokens without revenue (memes, narrative plays):
- Community strength (social followers, engagement rate, diamond hands %)
- Mindshare (% of crypto Twitter conversation)
- Liquidity moat (locked LP, wide CEX listings)
- Cultural staying power (Lindy effect — survived how many cycles?)
- Holder distribution (whales vs retail, Gini coefficient)

## How to Apply

1. Identify token category → Select appropriate framework(s)
2. Pull comparable tokens in same category
3. Calculate ratios for target + comparables
4. Determine relative positioning (percentile rank)
5. Assess qualitative factors that justify premium or discount
6. Present range: conservative / base / optimistic
