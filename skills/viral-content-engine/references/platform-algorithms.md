# Platform Algorithm Quick Reference (2025-2026)

## 小红书 Algorithm

**Discovery flow:** Post → Small pool test (200-500 views) → Engagement check → Expand or die
**Key signals (ranked):**
1. Save rate (收藏率) — #1 signal, >5% is excellent
2. Comment rate — Especially long comments and author replies
3. Share rate — Direct sharing to WeChat
4. Click-through rate on cover image — Determines initial pool performance
5. Completion rate — For video content, watch-through %
6. Dwell time — How long readers spend on the post

**Algorithm hacks:**
- Reply to EVERY comment in first 2 hours
- Ask a question in the post to drive comments
- Cover image quality > content quality for initial distribution
- Posting time: 7-9am, 12-2pm, 7-10pm (CST)
- Avoid: direct ads, external links, competitor brand names
- Carousel (图文) posts get 2-3x more saves than single images

## Twitter/X Algorithm

**Distribution:** For You feed is king (>80% of impressions for growth)
**Key signals (ranked):**
1. Reply velocity (first 30 min)
2. Retweet/Quote ratio
3. Bookmark rate (hidden signal, very powerful)
4. Profile clicks from tweet
5. Dwell time on tweet
6. Follower engagement rate

**Algorithm hacks:**
- First reply should come within 5 min (reply to yourself with context)
- Threads: first tweet must standalone as valuable
- Avoid: external links in tweet body (put in reply)
- Media tweets get slight boost but text-only can go mega-viral
- Quote tweets count less than native retweets since late 2024
- Posting time: 8-10am EST for US tech, 2-4pm for global
- Engagement pods/DM groups: use carefully, platform detects patterns

## LinkedIn Algorithm

**Distribution:** Follows a 3-stage cascade: connections → 2nd degree → wider
**Key signals (ranked):**
1. Dwell time (people reading the full post)
2. Comment quality and length
3. Comment velocity (first hour critical)
4. Shares to messages (private shares count heavily)
5. Post format (native content > links)
6. Profile authority (SSI score, connection quality)

**Algorithm hacks:**
- "See more" click is a massive signal → front-load value but create curiosity
- Comment on your own post immediately to seed discussion
- Avoid: external links (50-70% reach penalty), AI-looking formatting
- Personal stories with professional lessons = highest engagement format
- Engage with commenters' profiles afterward (triggers reciprocity)
- PDF carousels/documents get 2-3x reach vs text posts
- Posting time: Tue-Thu, 7-9am local time of your network

## 公众号 Algorithm

**Distribution:** Push to subscribers + algorithm recommendation (看一看)
**Key signals (ranked):**
1. Open rate (subject line determines this)
2. Read completion rate (读完率)
3. 在看 (Wow/Like) rate → pushes to friends' 看一看 feed
4. Share to Moments (朋友圈) rate
5. Share to Groups/Friends rate
6. Comment activity
7. Follower retention (unfollows after reading = negative signal)

**Algorithm hacks:**
- Title A/B testing is THE most important optimization
- First 3 paragraphs determine if people finish → hook hard
- 金句 at end drives 在看 clicks (people want to share the wisdom)
- Optimal length: 1500-2500 chars for broad reach, 3000-5000 for deep engagement
- Publishing time: 8-9pm (highest open rates), 7-8am (commute reading)
- Cover image: 2.35:1 ratio, text overlay works well
- Add a "转发到朋友圈" CTA before the fold

## Threads Algorithm

**Distribution:** Similar to Instagram's explore → interest-based discovery
**Key signals (ranked):**
1. Reply rate
2. Like velocity
3. Repost rate
4. Quote rate
5. Content relevance to user interests

**Optimization:**
- Conversational, approachable tone wins
- Hot takes perform well if backed with reasoning
- Threading (multi-post threads) is supported and boosted
- Cross-posting from Instagram audience helps initial distribution
- Hashtags are minimal (1-3 max)
