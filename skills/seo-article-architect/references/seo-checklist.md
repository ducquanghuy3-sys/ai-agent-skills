# Technical SEO Content Checklist

## On-Page SEO (Per Article)
- [ ] Title tag: 50-60 chars, primary keyword front-loaded
- [ ] Meta description: 150-160 chars, includes keyword + CTA
- [ ] URL: short, keyword-rich, hyphens only, no parameters
- [ ] H1: one per page, matches/reflects title tag
- [ ] H2-H3 hierarchy: logical, includes secondary keywords
- [ ] Primary keyword: in first 100 words, 0.5-1.5% density
- [ ] Image alt texts: descriptive, include keywords where natural
- [ ] Image file names: keyword-descriptive (not IMG_001.jpg)
- [ ] Image compression: WebP format, <100KB per image
- [ ] Internal links: 3-5 relevant internal links per 1000 words
- [ ] External links: 2-3 authoritative outbound links
- [ ] Schema markup: Article, FAQ, HowTo, or Review as applicable
- [ ] Canonical tag: self-referencing canonical
- [ ] Mobile responsive: content looks good on mobile viewport
- [ ] Page speed: target <3s load time

## Content Quality Signals
- [ ] E-E-A-T: Author bio present, credentials mentioned
- [ ] Freshness: current year/date referenced
- [ ] Comprehensiveness: covers subtopics that top-3 results cover
- [ ] Original value: adds something no other result provides
- [ ] Readability: Flesch score 60+ (English) or appropriate Chinese level
- [ ] Media: at least 1 image per 300 words, tables for comparisons
- [ ] No thin content: minimum 1000 words for ranking intent

## Featured Snippet Optimization
- [ ] Target question identified from "People Also Ask"
- [ ] Answer formatted in 40-60 words (paragraph snippet)
- [ ] Or: formatted as numbered/bulleted list (list snippet)
- [ ] Or: formatted as table with clear headers (table snippet)
- [ ] Answer immediately follows the question (as H2 or H3)

## Google-Specific
- [ ] Core Web Vitals: LCP <2.5s, FID <100ms, CLS <0.1
- [ ] Structured data: valid (test with Rich Results Test)
- [ ] Indexability: no noindex, robots.txt allows crawling
- [ ] HTTPS: secure connection
- [ ] Sitemap: article included in XML sitemap

## Baidu-Specific (Chinese Content)
- [ ] Baidu Webmaster tools: site verified
- [ ] 原创声明: original content declaration submitted
- [ ] 百度百科 link opportunity: assessed
- [ ] Mobile adaptation: MIP or responsive verified
- [ ] 熊掌号/百家号 syndication: planned
- [ ] Title: ≤30 Chinese characters
- [ ] Load speed: optimized for Chinese CDN
